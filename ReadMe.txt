How to lauch the Auto-Video-Games

1. Install Visual Studios 2017 in the default drive.
2. Make sure the program files are in the default drive "C:\CSE201\CSE 201.sln" /t:build /p:Configuration=Release"
3. Launch test.bat file (double click)


How to edit Auto-Video-Games (for developers)
1. Open files with Visual Studios or preferred editing software.
2. Make appropriate changes.
3. Push edited files to the Github repository.

The website will build and launch using the default browser
